# 📚 背词大师v0 (VocBuilder)

[👉 点击这里直接在线体验](https://zzzz1783.github.io/first1fdad/)

这是一个极简的、基于纯前端（HTML + Vanilla JS + CSS）打造的背单词网页工具。无需安装任何后端或数据库，所有数据均安全保存在浏览器本地。

## ✨ 核心功能

* **⚡ 极简单文件**：整个应用只有一个 `.html` 文件，支持完全离线运行。
* **🤖 AI 智能导入**：内置专属 Prompt，可配合“豆包”等大语言模型，将任何杂乱文本快速洗成标准词库并导入。
* **🧠 记忆管理**：支持“错题本”和“简单词”分类，自动记录练习历史与错误次数。
* **🎯 多种测试模式**：支持“随机/新词”复习模式以及“拼写测试”模式。
* **🔊 原生发音**：调用浏览器原生 `SpeechSynthesis` API 实现单词朗读。

## 🚀 如何使用

1.  **在线访问**：直接点击上方体验链接即可在浏览器中使用。
2.  **添加词库**：
    * 进入网页的“导入”页面，新建一个分区（Deck）。
    * 点击“复制标准化提示词”，发送给豆包。
    * 将包含你想背的单词的文章发送给 AI，将 AI 输出的规范格式粘贴回网页进行导入。

## 🛠 技术栈

* **前端**：HTML5, CSS3 (原生变量与 Flexbox/Grid 布局)
* **逻辑**：Vanilla JavaScript (无任何第三方依赖)
* **存储**：`localStorage` (数据持久化)

## ⚠️ 注意事项

* 请勿清理浏览器的本地缓存（LocalStorage），否则会导致词库和学习记录丢失。
* 建议定期在“导入与管理”页面使用“导出”功能备份你的个人词库。
